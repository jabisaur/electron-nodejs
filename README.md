Aulas de Análise e Desenvolvimento de Sistemas - Senac HUB ACADEMY - Campo Grande, MS - Turma 64.2025

Sob orientação do professor Thiago Lodi, foi desenvolvido um APP Desktop utilizando Electron e Nodejs.

Linguagens utilizadas: Javascript, HTML e CSS.

A princípio foi criado uma página de versões onde o usuário pode visualizar quais versões do Node, Electron e Chrome foram utilizadas. Depois da criação dessa página, teve a
criação de uma página de contador, onde o usuário pode contar seus cliques. Seguindo para a próxima página, foi criado uma calculadora com funções matemáticas aplicáveis.

Como continuação de um projeto CRUD feito em SQL, implementou-se a página Loja de Música, onde é utilizado banco de dados. O usuário consegue cadastrar e salvar os dados em tabelas de 
músicas, artistas, discos, estilos e gravadoras. A categoira de artista é dividida entre interprete e compositor, onde o artista pode ser ou interprete ou compositor ou ambos. Existe uma tabela de relação chamada musica_disco onde as musicas são relacionadas com os discos.
